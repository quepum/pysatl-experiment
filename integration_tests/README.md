# Интеграционные тесты параллелизации

Набор сценариев для проверки корректности, стабильности и надёжности параллельной реализации.

## Сценарии
1. **Корректность** — сравнение результатов работы параллельного и последовательного режимов
2. **Взаимодействие с БД** — проверка конфликтов при параллельном чтении
3. **Воспроизводимость** — многократный запуск одной и той же конфигурации эксперимента с проверкой идентичности результатов.
4. **Стресс-тест** — большое число задач и трудоёмкие задачи
5. **Устойчивость к сбоям** — сохранение результатов всех успешно выполненных задач при падении задачи отдельной задачи
