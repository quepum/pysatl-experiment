### Тест корректности (Power-эксперимент)

**Цель проведения теста**:  
Проверка идентичности результатов, полученных в параллельном и последовательном режимах выполнения одного и того же эксперимента.

**Конфигурация**:  

| Параметр                | Значение |
|-------------------------|-------|
| Тип эксперимента        | `power` |
| Гипотеза                | `normal` |
| Статистические критерии | KS, AD, SW |
| Размеры выборок         | 100, 200, 300, 400, 500 |
| Уровни значимости       | 0.01, 0.05, 0.1 |
| Альтернативы            | Gamma(0.5, 1.0), Weibull(1.0, 2.0) |
| Monte Carlo Count       | 1 000 |
| Общее число задач       | **90** |
| Режим выполнения        | Параллельный / Последовательный |

**Результаты выполнения**:  

- **Статус завершения**: Успешно (оба режима)  
- **Общее время выполнения**:  
  - Последовательный режим: **806.33 секунд**,  
  - Параллельный режим: **482.01 секунд**   
- **Ошибки СУБД**: Не выявлено  
- **Число обработанных задач**: **90 из 90** (в обоих режимах)  
- **Сравнение результатов**:  
  - Все 90 записей в таблице `power` идентичны в обоих запусках на одной и той же выборке данных.

**Вывод**:  

Тест подтвердил полную корректность параллельной реализации: результаты, полученные в параллельном режиме, идентичны результатам последовательного выполнения. При этом достигнуто значительное ускорение: **806.33 / 482.01 ≈ 1.67×**. Это доказывает, что параллелизация не вносит ошибок в вычисления и является безопасной для существующей логики эксперимента.
