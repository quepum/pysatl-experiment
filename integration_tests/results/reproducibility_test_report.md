### Тест воспроизводимости (Time Complexity)

**Цель проведения теста**:  
Проверка детерминированности параллельной реализации: выполнение одного и того же эксперимента несколько раз подряд должно давать идентичные результаты на одной и той же выборке данных.

**Конфигурация**:  

| Параметр                | Значение |
|-------------------------|---|
| Тип эксперимента        | `time_complexity` |
| Гипотеза                | `normal` |
| Статистические критерии | KS |
| Размеры выборок         | 100, 200 |
| Monte Carlo Count       | 500 |
| Общее число задач       | 2 |
| Режим выполнения        | Параллельный |
| Число запусков          | 5 |

**Результаты выполнения**:  

- **Статус завершения**: Успешно (все 5 запусков)  
- **Статистические характеристики времени выполнения**:
  - Математическое ожидание: **8.78 с**  
  - Среднеквадратичное отклонение: **0.78 с**
- **Ошибки СУБД**: Не выявлено  
- **Число обработанных задач**: **2 из 2** (в каждом запуске)  
- **Сравнение результатов**:  
  - Все 5 запусков дали идентичные результаты по результатам отчётов.

**Вывод**:  

Тест подтвердил полную воспроизводимость параллельной реализации: 5 последовательных запусков одного и того же эксперимента дали одинаковые результаты.
